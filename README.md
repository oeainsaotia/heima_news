# heima_news

一新闻后端项目，包括app端，自媒体端和管理员端。app端用户可以浏览新闻并使用关键字搜索文章，自媒体端可以保存素材和发布文章，管理员可以对用户和文章进行管理。

下载到本地：
```cmd
git clone https://github.com/oeainsaotia/heima_news.git
```
